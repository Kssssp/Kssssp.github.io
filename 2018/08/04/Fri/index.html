在开始编写代码之前首先了解一下要用到的两个python库：

  • requests Requests 是用Python语言编写，基于 urllib，采用 Apache2 Licensed 开源协议的 HTTP 库。它比 urllib 更加方便，可以节约我们大量的工作，完全满足 HTTP 测试需求。
  • BeautifulSoup 简单来说，Beautiful Soup是python的一个库，最主要的功能是从网页抓取数据。

这两个库都十分便捷且强大，能提供我们编写爬虫代码时需要的的绝大部分功能
